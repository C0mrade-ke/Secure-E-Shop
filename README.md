#  Secure E-Shop Checker

**Secure E-Shop Checker Lite** is a simple web-based tool that helps users and small e-commerce businesses identify potentially insecure websites. It focuses on basic **web security indicators** and educates users on safe online shopping practices.

---

## ðŸ”¹ Problem

Many e-commerce websites are **not fully secure**, exposing users to risks such as:

- Stolen passwords and payment details  
- Fake or phishing websites impersonating popular brands  
- Lack of encryption or weak security practices  

Users often **cannot tell if a website is safe**, especially when they are shopping online for the first time.

---

## ðŸ”¹ Solution

Secure E-Shop Checker Lite allows users to:

1. Enter a website URL  
2. Automatically check basic **security indicators** such as:

   - HTTPS usage  
   - Suspicious characters in the URL  
   - E-commerce-related keywords (`/checkout`, `/login`, `/cart`)  

3. Receive a **security score and color-coded result**:  

   -  SAFE  
   -  CAUTION  
   -  UNSAFE  

4. View **educational notes** explaining the risks and recommendations for safe online shopping.
